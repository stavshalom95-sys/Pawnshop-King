Audio drop folder
=================

Drop audio files here (WAV/OGG/MP3) and they replace the built-in
synthesized placeholders on the next play session:

    Click.wav    - UI button press
    Accept.wav   - deal closed / purchase
    Reject.wav   - deal refused / customer storms out
    Music.ogg    - looping background track (noir/ambient)

Names must match exactly (extension can be .wav/.ogg/.mp3). Until a
Music file exists, a quiet synthesized drone pad stands in. Music
loops, so use a track prepared for seamless looping.
