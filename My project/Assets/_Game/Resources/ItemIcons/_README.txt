Item art drop folder
====================

Drop PNG sprites here and they appear in the game immediately —
import settings are applied automatically.

Naming (first match wins):

1. Exact item display name, e.g.
     Military Field Watch.png
     Gold Ring.png
     Recent-Model Smartphone.png

2. Category name (used for every item of that category without its own file):
     Watches.png
     Jewelry.png
     Electronics.png
     MusicalInstruments.png
     RetroCollectibles.png
     AntiquesCurios.png
     LuxuryAccessories.png
     ToolsPracticalGoods.png

3. No file -> the built-in procedural glyph is shown.

Tips: square images with transparent backgrounds look best in the icon
chip; anything roughly 128-256 px is plenty at UI size.
